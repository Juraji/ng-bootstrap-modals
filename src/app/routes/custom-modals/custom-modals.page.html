Custom modals

Have any component be rendered in a modal-style pop-up.

You can use Bootstrap's modal layout, like in example.modal.html below, or use your own component.
Say for instance a file-browser, or another website, if you so please.

Try it out!

  • Resolved status: {{modalResolve$ | async}}
  • Dismissed status: {{modalDismiss$ | async}}
  • Completed status: {{modalComplete$ | async}}
