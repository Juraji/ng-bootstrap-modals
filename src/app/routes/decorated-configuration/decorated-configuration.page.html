Decorated configuration

New and improved per modal configuration!

I was struggling with having the modal configuration decoupled from the actual modal component.
For instance a modal, which is designed to be large or fullscreen, had to be configured like this on every call.

Introducing Decorated Configuration.
For each of the configuration options there is now a convenient decorator which can be put on the modal component class.

The decorators are to be set on class-level, like Angular's @Component(...) decorator. Check the examples below!
All decorators stack, with the last defined decorator of the same type taking precedence over the previous decorators!

Configuration is applied using a merging strategy in the following order:

  1. Default configuration
  2. Root module configuration (optionally supplied when calling ModalsModule.forRoot(...))
  3. Decorated configuration (From top to bottom)
  4. Instance configuration (optionally supplied when calling Modals.open(...))
NG Bootstrap Modals is fully compatible with Bootstrap 4 and 5. This means some options are for a particular version only. Not to worry, I've noted that down below as well.
Property Decorator Behaviour
scrollable @ModalScrollableEnabled() Enable modal scrolling (default)
scrollable @ModalScrollableDisabled() Disable modal scrolling
centered @ModalCenteredEnabled() Enable vertically centering modal (default)
centered @ModalCenteredDisabled() Disable vertically centering modal, modal floats to top of page (with some margin on top)
keyboard @ModalKeyboardEnabled() Enable keyboard bindings, like escape to close etc. (default)
keyboard @ModalKeyboardDisabled() Disable keyboard bindings
backdrop @ModalBackdropEnabled() Enable modal backdrop (default)
backdrop @ModalBackdropDisabled() Disable modal backdrop
backdrop @ModalBackdropStatic() Set static modal backdrop, does not respond to clicks
ariaLabelledBy @ModalAriaLabelledBy() Set ARIA labelled by property (for screen readers and stuff)
windowClass @ModalWindowClass() Add to modal window class (space separated)
zIndex @ModalZIndex() Set modal z-index
backdropClass @ModalBackdropClass() Add to modal backdrop class (space separated)
size @ModalSizeSm() Set modal size to small
size @ModalSizeMd() Set modal size to medium
size @ModalSizeLg() Set modal size to large
size (Bootstrap@5+ only) @ModalFullScreen() Set modal size to fullscreen
size (Bootstrap@5+ only) @ModalFullScreenSmDown() Set modal size to fullscreen, only on screens smaller than sm break point
(combines with sm, md and lg decorators)
size (Bootstrap@5+ only) @ModalFullScreenMdDown() Set modal size to fullscreen, only on screens smaller than md break point
(combines with sm, md and lg decorators)
size (Bootstrap@5+ only) @ModalFullScreenLgDown() Set modal size to fullscreen, only on screens smaller than lg break point
(combines with sm, md and lg decorators)
size (Bootstrap@5+ only) @ModalFullScreenXlDown() Set modal size to fullscreen, only on screens smaller than xl break point
(combines with sm, md and lg decorators)
size (Bootstrap@5+ only) @ModalFullScreenXxlDown() Set modal size to fullscreen, only on screens smaller than xxl break point
(combines with sm, md and lg decorators)
