Shade Modals

The Blocking Shade modal is designed to block a user from the UI during a long running task.

It has support for displaying progress, by supplying an observable which emits the progress from 0 until 100.
When nothing is emitted from the observable, or it is empty, the shade will display an indeterminate progress bar.
